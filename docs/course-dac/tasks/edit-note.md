# Пользовательский сценарий "Редактирование заметки"

## Действующие лица

1. Пользователь

2. Приложение

3. Бэк

4. База данных

## Предварительные условия

Пользователь должен находиться на главном экране.

## Выходные условия

В системе внесены изменения в выбранную заметку.

## Основной сценарий

1. Пользователь нажимает на выбранную заметку.

2. Приложение запрашивает у Бэка данные выбранной заметки.

3. Бэк запрашивает данные заметки из базы данных.

4. Бэк возвращает приложению данные заметки.

5. Приложение отображает информацию заметки.

6. Пользователь нажимает кнопку **Редактировать**.

7. Приложение открывает окно с формой редактирования заметки.

8. Пользователь вносит изменения в заметку и нажимает кнопку **Сохранить**.

9. Приложение отправляет запрос Бэку на сохранение измененной заметки.

10. Бэк сохраняет заметку в базе данных.

11. Бэк возвращает приложению ответ об успешном сохранении заметки.

12. Приложение открывает пользователю уведомление «Изменения сохранены!» и отображает обновлённую информацию заметки.

## Альтернативный сценарий "Возвращение на главный экран"

1. Пользователь нажимает на выбранную заметку.

2. Приложение запрашивает у Бэка данные выбранной заметки.

3. Бэк запрашивает данные заметки из базы данных.

4. Бэк возвращает приложению данные заметки.

5. Приложение открывает пользователю информацию заметки.

6. Пользователь нажимает кнопку **Мои заметки**.

7. Приложение возвращает интерфейс главного экрана со всеми заметками.

## Альтернативный сценарий "Отмена сохранения"

1. Пользователь нажимает на выбранную заметку.

2. Приложение запрашивает у Бэка данные выбранной заметки.

3. Бэк запрашивает данные заметки из базы данных.

4. Бэк возвращает приложению данные заметки.

5. Приложение открывает пользователю информацию заметки.

6. Пользователь нажимает кнопку **Редактировать**.

7. Приложение открывает окно с формой редактирования заметки.

8. Пользователь вносит изменения в заметку.

9. Пользователь нажимает кнопку **Отмена**.

10. Приложение закрывает форму редактирования и возвращает пользователя к просмотру заметки без сохранения изменений.
